## Smb_Nfs

Script sencillo para enumerar shares conformado otros scripts conocidos como smbmap, smbcliet, nmap y enum4linux

## Sistema Operativo

Parrot OS 4.11.2

## Intalacion

git clone https://github.com/Anonimo501/Smb_Nfs.git

cd Smb_Nfs

chmod +x smb_nfs_shares.sh

## Ejecucion

./smb_nfs_shares.sh
